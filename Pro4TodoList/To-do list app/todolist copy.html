To-Do List

    About the Project ▲

    This project is a simple To-Do List application created as part of the "30 Days 30 JavaScript Projects" challenge. It's designed by Tushar Aher using HTML, CSS, and JavaScript. The application allows users to add new tasks and remove completed ones, providing an efficient way to manage tasks and stay organized. Embrace productivity and organization with this sleek and user-friendly to-do list application.
